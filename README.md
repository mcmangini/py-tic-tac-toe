# py-tic-tac-toe

Tic-toc-toe on the command-line, written in Python.

## Usage

```
git clone https://github.com/mcmangini/py-tic-tac-toe.git
cd src
python3 tictactoe.py
```

## Game Modes

- Two-Player
    - Play against another person.
- One-Player
    - Play against a CPU.
- Zero-Player
    - Have two CPUs play each other.
